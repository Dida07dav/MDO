Python поддерживает динамическую типизацию, то есть тип переменной определяется только во время исполнения. 
Поэтому вместо «присваивания значения переменной» лучше говорить о «связывании значения с некоторым именем.»
К примитивным типам в Python относятся булевый, целое число произвольной точности, число с плавающей запятой и комплексное число
